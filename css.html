CSS

CSS which stands for Cascading Style Sheets, was developed in 1996 and describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. CSS technically is not a programming language but rather a stylesheet/styling language that is used for website layout and design. CSS can also be extended with SASS (Syntactically Awesome Style Sheets) as well as LESS (Leaner Style Sheets) which are both language extensions for CSS.
