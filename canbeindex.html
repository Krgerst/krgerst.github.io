welcome to project 0

an informational site regarding c.i.t. 82

Hi all,

I've created this site to showcase some of the topics that we have covered throughout this course. Please feel free to have a look around and enjoy!

some of what this site covers

CIT 82 intro to Web Development with professor Waller, which has run from August 12th until today (October 10th), has been a really great class and we have covered such a wide range of topics on top of the basics of web development; from the inner workings of the web, cloud storage and even subjects like machine learning. All of these topics leading up to our final project, Project 0 in which we were tasked with creating this beautiful website to showcase just some of the work that we have done over the course of the semester, or a site about ourselves. This site was created using a combination of CSS, HTML and Bootstrap. Being such a hand-on assignment, Project 0 really gave me a better feeling of how the three work hand-in-hand. This was a fun project and I believe that I have learned a lot from it.
